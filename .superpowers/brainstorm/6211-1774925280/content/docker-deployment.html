Key Docker Decisions

Compose Profiles

Single docker-compose.yml with dev and prod profiles instead of separate files. Cleaner, less drift.

Health Checks

Every service gets a /health endpoint. Required for Traefik routing and for depends_on with condition: service_healthy.

External Network

Production joins an external Docker network (e.g. pangolin_net) so Traefik can reach it. No port publishing needed.

Python Dockerfile

Simple: python:3.12-slim, install uv, copy lockfile, install deps, copy app. No multi-stage Rust compilation nightmare.
